# 介绍

turbo 的诞生旨在为基于 [tornado](http://tornado.readthedocs.org/en/stable/) 和 [mongodb](https://www.mongodb.org/) 的应用开发提供**快速构建**，**便于扩展**，**易于维护**的最佳实践方案。turbo 包含以下特性

- 简易的 orm (基于 mongodb ) 
- 快速构建 rest api
- 易于扩展和维护的 app 结构
- 灵活的 log 
- 简单可以增强的 session
